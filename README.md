# Madown

Madown translates markdown into HTML.

## Keywords

- header
- unordered list
- link
- image
- emphasis
- italic text
- quote
- code block

## Build

### generate makefile

```bash
$ cd cmake-build-debug
$ cmake ..
```

### build library

you will get `libMaDown.a`

```bash
$ make Madown
```

### build example

```bash
$ make example
```
